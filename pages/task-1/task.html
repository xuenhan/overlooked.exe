JAWS (screen reader)

JAWS (Job Access With Speech) is a screen reader computer program for Microsoft Windows that allows blind and visually impaired users to read the screen either with a text-to-speech output or by a refreshable Braille display.

JAWS is produced by Freedom Scientific, a company headquartered in the United States.

History

JAWS was originally released in 1989 by Henter-Joyce. It became one of the most widely used screen readers in the world. Over time, it expanded to support web browsing, office software, and other digital interfaces.

Features

JAWS provides speech and Braille output for most Windows applications. It allows users to:

  • Navigate web pages using keyboard shortcuts
  • Read documents and emails
  • Access menus and dialog boxes

Users can stop speech output at any time by pressing the Control key .

Compatibility

JAWS works with Microsoft Office, web browsers, and many other applications. It integrates with accessibility APIs provided by Windows.
